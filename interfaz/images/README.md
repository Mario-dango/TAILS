# 🖼️ Recursos Visuales (Assets)

Este directorio es el repositorio estático de todos los elementos multimedia utilizados por la Interfaz Gráfica (Capa View) de T.A.I.L.S.

## Contenido
* **Iconos (`.png`, `.ico`)**: Imágenes renderizadas que se inyectan dinámicamente en los botones de la interfaz mediante la función `set_btn_icon` de la vista.
* **Modo Kawaii**: Todos los gráficos alojados aquí responden al "Modo Kawaii" (toggle visual de la UI). Si este modo se desactiva desde el menú superior, la interfaz dejará de renderizar los elementos de esta carpeta para adoptar un formato de texto minimalista.

*Nota: Asegúrese de mantener las proporciones cuadradas (ej. 32x32, 64x64) al agregar nuevos iconos para evitar deformaciones en los layouts de PyQt5.*
